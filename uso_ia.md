# Uso de Inteligência Artificial no Projeto

## Ferramentas utilizadas
- Claude (Anthropic) — assistente de IA

## Como foi utilizada
- **Organização de ideias**: Auxiliou na estruturação inicial do projeto e na definição das seções do relatório.
- **Geração dos dados simulados**: Os dados de telemetria do CSV foram criados com apoio da IA para garantir coerência com um cenário realista de missão espacial.
- **Revisão de texto**: Revisão ortográfica e de clareza nos comentários do código e no relatório.

## O que foi desenvolvido pela equipe (sem IA)
- Toda a lógica de diagnóstico (regras IF/ELIF/ELSE e expressões booleanas)
- A implementação da regressão linear (mínimos quadrados do zero)
- A estruturação das funções e a escolha das estruturas de dados
- A análise e interpretação dos resultados

## Validação crítica realizada
- Cada sugestão da IA foi testada manualmente antes de ser incorporada.
- O código foi executado e validado linha a linha pela equipe.
- As fórmulas matemáticas da regressão foram verificadas manualmente com os dados do CSV.
- As regras lógicas foram testadas com valores de borda (reserva = 20%, 50%; radiação = 0.7, 1.0 Sv).
